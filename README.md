### Advanced Machine Learning: Регрессия, Классификация и Основы CV

В данных материалах исследуются фундаментальные алгоритмы обучения с учителем, методы регуляризации для борьбы с переобучением, а также базовые техники обработки изображений и многоклассовой классификации.



#### Computer Vision: Основы обработки изображений и классификации
* **Многоклассовая логистическая регрессия:** Применение функции SoftMax для классификации визуальных образов.
* **Фильтрация и детектирование границ:** Реализация сверток с использованием операторов Собеля (Sobel X/Y) для выделения градиентов и структурных элементов изображения.
* **Математическая морфология:** Применение фильтров для сегментации и анализа объектов на изображениях.

#### KNN: Метод ближайших соседей
* **Алгоритмический анализ:** Сравнение классического KNN, метода ближайших центроидов и классификации с учетом различных метрик расстояния (Махаланобиса, Хэмминга).
* **Оптимизация гиперпараметров:** Исследование влияния количества соседей ($K$) на точность модели и поиск оптимального значения для минимизации ошибки на тестовой выборке.
* **Масштабирование данных:** Анализ необходимости StandardScaler для корректной работы метрических алгоритмов.

#### L1 & L2: Регуляризация и обобщающая способность
* **Борьба с переобучением:** Сравнительное исследование Ridge (L2) и Lasso (L1) регрессий.
* **Отбор признаков:** Демонстрация свойства Lasso-регуляризации занулять коэффициенты малозначимых признаков.
* **Полиномиальная регрессия:** Анализ роста сложности модели и использования регуляризации для стабилизации предсказаний при высоких степенях полинома.

#### RegresClass: Метрики качества и линейные модели
* **Линейная классификация:** Применение логистической регрессии и метода опорных векторов (SVC) для разделения классов.
* **Бизнес-ориентированные метрики:** Оценка регрессионных моделей через MSE, MAE и R² с анализом стоимости ошибок (Overstock vs Stockout) в реальных задачах.
* **Диагностика моделей:** Построение гистограмм ошибок и анализ ROC-кривых для оценки разделяющей способности классификаторов.

#### SoftMax: Вероятностная классификация
* **Логиты и вероятности:** Практическая реализация функции SoftMax для преобразования выходов дискриминантных функций в вероятностное распределение.
* **Температурное сглаживание:** Исследование влияния параметра температуры ($\tau$) на "уверенность" модели и распределение вероятностей между классами.
* **Визуализация отступов:** Анализ расположения объектов относительно разделяющих гиперплоскостей на классическом датасете Iris.
